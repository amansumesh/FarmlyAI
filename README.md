# Farmly AI - Multilingual Voice-First Smart Farming Assistant

**Farmly AI** is an AI-powered agriculture support platform designed to help Indian farmers make faster and smarter decisions using **voice-first interaction**, **offline crop disease detection**, and **real-time cloud intelligence**. It provides personalized crop advisory, market insights, and government scheme guidance through an easy-to-use Progressive Web App (PWA).

The main goal of Farmly AI is to solve critical farming challenges like crop disease losses, lack of expert access, language barriers, and poor internet connectivity by delivering agricultural expertise directly to farmers in a simple and accessible way.

---

## Project Overview

Farmly AI is built as an **end-to-end farming ecosystem** where farmers can access everything from disease detection to market prices and government schemes in one platform.

The system allows farmers to:

- Detect crop diseases using AI (offline mode supported)
- Get advanced cloud-based treatment recommendations
- Use multilingual voice interaction for easy accessibility
- Access real-time weather and market price intelligence
- Check government schemes and eligibility guidance
- Work even in low internet or no internet environments
- Store and sync data securely when online

The platform is designed to be scalable and usable for millions of farmers across India.
